# 📧 Email Metrics Dashboard

An interactive Tableau dashboard for analyzing email campaign performance,
including sent, open, and click-through rates across different countries.

## 🔗 Live Dashboard
[View on Tableau Public](https://public.tableau.com/app/profile/kristina.vasylenko/viz/EmailMetrics_17399142258870/EmailMetrics)

## 📊 Key Metrics
- **Sent Count** – total emails sent per day
- **Open Rate** – share of opened emails
- **Click Rate** – share of emails with link clicks
- **Account Count** – active accounts per day

## 🛠 Tools
- **SQL** – BigQuery (Google Cloud)
- **Visualization** – Tableau Public
- **Data** – e-commerce dataset (DA database)
